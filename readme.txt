IG MoonPay Pack - README

Files included:
- shop.html          => Micro-shop page (single page) you can host and link from Instagram bio.
- link_generator.html => Mobile-friendly MoonPay link generator. Use it on your phone to create DM-ready links.
- sample_qr_links.txt => Example MoonPay links and Google Chart QR links for common amounts.

How to use:
1) Upload shop.html to any static host (Netlify, Vercel, GitHub Pages) or open locally.
2) Use link_generator.html on your phone to quickly create payment links and paste them into Instagram DM.
3) Use sample QR links if you want printable QR codes for fixed price items.
